Banco - André Nunes - Segundo módulo 

crescer verá principalmente oracle e SQL server.

sequencia desempenho em algumas vezes maior que identity.

4 tipos de constraints: primary key, unique(pode ser opicional mas se existir tem que ser única), foreign key, check(um dado
precisa respeitar uma condição).

char(n) - varchar(n) - útil apenas para aplicações globais. Guarda o dobro de espaço

ddl - data definition language (create table, drop table)
dml - data manipulation language (insert, update, delete)

count, min, max, sum, avg - Principais funções

distinct elimina linhas da consulta que são iguais.

INNER join - ambas as tabelas com um registro existente.
LEFT join or RIGHT join - força exibir todos os registros de uma tabela, mesmo que seja null na outra

EXISTS or in(subquery) - parecidos mas diferentes.

union - armazena em memória, ordena e elimina os duplicados (distinct)

union all - projeta e não tira os iguais
